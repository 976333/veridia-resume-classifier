# Veridia Resume Classifier – AI/ML Internship Task (Option B)

An end-to-end resume classification system that categorizes resumes into job domains using NLP and machine learning.

## 📌 Dataset
- Source: [Kaggle Resume Dataset](https://www.kaggle.com/datasets/snehaanbhawal/resume-dataset)
- Contains 2,484 resumes across 25+ categories.
- **To use**: Download `Resume Dataset.csv`, rename to `resume_dataset.csv`, and place in `data/`.

## 🛠️ Tech Stack
- **Backend**: Python, Flask
- **ML**: scikit-learn, pandas, NLTK, TF-IDF
- **PDF Handling**: PyPDF2
- **Frontend**: HTML, CSS, JavaScript (Tailwind-inspired dark UI with animated background)
- **Deployment**: Local Flask server
